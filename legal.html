Finalidad del Sitio Web

Climate Checker es un sitio web que permite consultar información meteorológica actual de distintas ciudades, agregando datos de múltiples fuentes públicas (Open-Meteo, wttr.in, WeatherAPI, OpenWeatherMap). Dispone de un panel de administración protegido por autenticación para la gestión de datos maestros.

Propiedad Intelectual

Todos los contenidos del sitio web (texto, diseño, código) son originales del titular, salvo que se indique lo contrario. Los datos meteorológicos mostrados pertenecen a sus respectivas fuentes y se utilizan bajo sus términos de servicio.

Política de Privacidad

De acuerdo con el Reglamento (UE) 2016/679 (RGPD) y la Ley Orgánica 3/2018 (LOPDGDD), se informa sobre el tratamiento de los datos personales.

Datos Recogidos

  • Datos de navegación: dirección IP, ubicación aproximada (si el usuario concede permiso de geolocalización), consultas de ciudades.
  • Datos de registro (admin): dirección de correo electrónico y contraseña (almacenada de forma segura por Supabase Auth, nunca en texto plano).

Finalidad del Tratamiento

  • Mostrar información meteorológica de la ciudad solicitada.
  • Gestionar el acceso al panel de administración.
  • Mejorar el servicio mediante análisis agregado de búsquedas.

Base Legal

  • Consentimiento del usuario (geolocalización, registro).
  • Ejecución de un servicio solicitado por el usuario (consulta meteorológica).
  • Cumplimiento de obligaciones legales.

Destinatarios de los Datos

Los datos se almacenan en Supabase (proveedor de base de datos y autenticación en la nube con servidores en EE.UU. y UE). Supabase cumple con el RGPD mediante las Cláusulas Contractuales Tipo (SCC). No se ceden datos a terceros salvo obligación legal.

Derechos del Usuario

El usuario puede ejercer sus derechos de acceso, rectificación, supresión, limitación, portabilidad y oposición mediante correo electrónico a la dirección indicada en el aviso legal. También puede presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD).

Conservación de Datos

Los datos de los usuarios registrados se conservan mientras la cuenta esté activa. Los datos de navegación anónimos se conservan por un período máximo de 12 meses.

Política de Cookies

Este sitio web utiliza únicamente cookies técnicas necesarias para el funcionamiento del sistema de autenticación (panel de administración).

Cookies Utilizadas

  • sb-*-auth-token: cookie de sesión de Supabase Auth, necesaria para mantener la sesión iniciada en el panel de administración. Se elimina al cerrar sesión o cerrar el navegador.

No se utilizan cookies de terceros ni cookies de seguimiento o publicitarias.

Configuración del Navegador

El usuario puede configurar su navegador para rechazar las cookies, si bien esto impedirá el uso del panel de administración. La consulta meteorológica pública no requiere cookies.
